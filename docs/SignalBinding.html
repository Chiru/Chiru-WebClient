Class: SignalBinding

SignalBinding

new SignalBinding(signal, listener, isOnce, listenerContext, priority)

Object that represents a binding between a Signal and a listener function.
- This is an internal constructor and shouldn't be called by regular users.
- inspired by Joa Ebert AS3 SignalBinding and Robert Penner's Slot classes.
Parameters:
Name Type Argument Description
signal Signal Reference to Signal object that listener is currently bound to.
listener Function Handler function bound to the signal.
isOnce boolean If binding should be executed just once.
listenerContext Object <optional>
 Context on which listener will be executed (object that should represent the `this` variable inside listener function).
priority Number <optional>
 The priority level of the event listener. (default = 0).
